# WorldEdit

## 道具の説明
```
斧
→ブロックを右クリ＆左クリで始点、終点を決めて範囲を指定

羽
→ブロックを左クリしたあとで右クリをすると視点の先のブロックを左クリしたブロックに置き換えられる

コンパス
→右クリすることで自分の下にガラスを置ける。空中作業に適している

棒
→ブロックを左クリでブロックのIDを取得

ハサミ
→ブロックを右クリでインベントリにコピー(コマブロやチェストは中身もコピーできる)

シャベル
→右クリで視点の先に砂の球体を召喚

ブラシ
→右クリで視点の先に不明ブロックの球体を召喚
```

## チャット系
```
\\set ブロックID
→指定範囲をブロックで埋める

\\outline ブロックID
→指定範囲の外側だけをブロックで埋める

\\kaiten
→指定範囲のブロックを90度回転させる

\\copy 
→指定範囲をコピー

\\cut
→指定範囲を切り取ってコピー

\\rotate -角度(0 , 90 , 180 , 270)
→コピーした範囲を貼り付けるときに何度回転させるかを決める

\\paste
→コピーした範囲を貼り付け

\\r 半径(数字) ブロックID
→自分を中心に指定した半径の円を作成(円周)

\\c 半径(数字) ブロックID
→自分を中心に指定した半径の球体(外側)を作成
```
